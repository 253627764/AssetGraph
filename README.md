# AssetGraph

Automate asset generating from import to AssetBundlize!

AssetGraph is a graph editor for generate Unity Assets from any resources.

素材からアセットバンドルまで、素材をブチ込むだけでOK

##Usage
###1.Add "node"
右クリックでノードを置いていく。
ImportとかBundlizeとか。

###2.Connect them
繋ごう。
プレビューでいろいろ見れるな。

###3.Set Parameters
インポートして、素材をどうしようか。設定が反映されるっぽいぞ。

###4.Build!
ビルドが終わって、チン！ 
簡単でしょう？
